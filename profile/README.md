# KiviBot 

开源、高效、跨平台的 QQ 机器人框架

**仓库和文档**

- 使用文档：https://beta.kivibot.com/
- 框架核心仓库：https://github.com/KiviBotLab/KiviBot
- 脚手架仓库：https://github.com/KiviBotLab/cli
- 文档仓库：https://github.com/KiviBotLab/docs
